Jasper’s World — Minimal Guardrails Pack

This pack contains ONLY guardrails required to prevent:
- Engine deadlocks or hangs
- Camera/parallax corruption
- Lifecycle leakage across states or levels
- Catastrophic rendering artifacts

Also included:
- Coding documentation guardrail (to preserve solved decisions)
- Two optional helpers safe for early implementation

Anything not in this pack is intentionally omitted to preserve creative freedom during early design.
